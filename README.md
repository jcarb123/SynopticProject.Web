# Synoptic Project Web Application

# Built With
- Frontend: Vue.js
- Cloud Services: Azure Static Web app
- CI/CD: GitHub Actions Workflow


# Local Setup
Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

Start the server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev

# bun
bun run dev
```
